# Online Shopping Project in Node.js and MySQL

## Project Overview
This is a web-based online shopping system built using Node.js and MySQL. The platform has separate layouts for admin and users. Users can browse products, register, login, and make purchases. Admins can manage products and view orders.

## Features

### Admin Features
- Add, Edit, and Delete Products
- View Orders

### User Features
- Register and Login
- Browse Products
- Purchase Products
- Check Purchase Status

## Technology Stack
- Node.js: Backend runtime environment
- MySQL: Database for storing user and product data
- HTML, CSS, Bootstrap: Frontend design and responsive layout
- JavaScript: Client-side scripting
